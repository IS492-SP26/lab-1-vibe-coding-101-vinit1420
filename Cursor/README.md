## Ping-Pong Game

A two-player Ping-Pong game built with Python and Pygame.

### Setup

```bash
# If 'pip' isn't recognized, use one of these instead:
py -m pip install -r requirements.txt
# or
python -m pip install -r requirements.txt
```

### Run

```bash
py ping_pong.py
# or
python ping_pong.py
```

### Controls

- **Player 1 (left paddle):** `W` – up, `S` – down  
- **Player 2 (right paddle):** `↑` – up, `↓` – down  
- **ESC** – quit  
- First to 10 points wins. Press **SPACE** after game over to play again.
